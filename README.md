# 🤖 AI Email Automation Agent (LangGraph + LLMs)

An AI-powered email automation system that allows users to send emails using natural language commands.
This project uses LangGraph to orchestrate LLMs, database queries, and automated email delivery via Nodemailer.

---

## 📌 Features

* 🤖 AI Agent for intelligent task automation
* 🗣️ Natural language command support
* 🧠 LLM-based email content generation
* 🗄️ Database-driven recipient lookup
* 📩 Automated email sending using Nodemailer
* ⚡ Fast and scalable workflow execution

---

## 🧠 How It Works

1. User provides a command in natural language
   → "Send an email to Rahul about project update"

2. AI agent processes the input
   → Extracts recipient name and topic

3. Database lookup is performed
   → Retrieves recipient email address

4. LLM generates email content
   → Creates subject and body

5. Nodemailer sends the email
   → Email successfully delivered

---

## 🔄 Workflow

User Input → AI Agent → Database → LLM → Email Generator → Nodemailer → Sent ✅

---

## 🛠️ Tech Stack

* Frontend: React.js (optional UI)
* Backend: Node.js / Express
* AI Orchestration: LangGraph
* LLMs: gemini / Mistral
* Email Service: Nodemailer
* Database: MongoDB
